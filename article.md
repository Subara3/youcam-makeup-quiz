## 同じシーンに、違う差分

![](https://static.zenn.studio/user-upload/217b75f051e4-20260614.png)

同じ会議室、同じ人。違うのは今日の気分だけ。それをリップの色温度、チークの有無、眉の強さで描き分けているつもりですが、さて、どうでしょうか。ピンとくるでしょうか。よーくよーく見てみるとメイクが少しずつ違うことに気が付きますね。

## どうやって作っているか

差分は全部で56枚（14シーン × 4気分）あります。さすがに手作業だと日が暮れるので、YouCam オンラインエディターを Playwright で動かして用意しました。

ちょっと気に入っているのが、色の選び方です。エディタのカラーパレットは、ひとつひとつの色玉が自分の色（`background-color`）を持っています。なので全部読み取って、「今日はこの赤にしたい」と決めた色にいちばん近い玉を、自動で選んでクリックさせています。

```js
// パレットの色玉から、狙った色にいちばん近いものを選ぶ
const balls = [...document.querySelectorAll('[class*="color-ball_singleColor"]')];
let best, min = Infinity;
for (const el of balls) {
  const [r, g, b] = getComputedStyle(el).backgroundColor.match(/\d+/g).map(Number);
  const d = (r - tr) ** 2 + (g - tg) ** 2 + (b - tb) ** 2; // 狙いの色との距離
  if (d < min) { min = d; best = el; }
}
best.click();
```

感情ごとに「はしゃぎはピンクのグロス」「いらだちは深い赤のマット」「しょんぼりはくすんだ色」と色と質感を決めて、あとはシーン写真を差し替えながら順番に乗せていくだけです。複数人が写っているシーンだと、たまにメイクが別の人の顔に乗ってしまうことがあって、何度かやり直しました（会議中、同僚が知らないうちにメイクされていた）。

ちなみに今回は色と質感までは指定できましたが、濃さの数値まではWeb版だと触れませんでした。だから差はかなり繊細です。でもそれが、かえって「言われないと気づかない」感じになって、結果的にこのクイズに合っている気がしています。

## こんなのも面白そう、という話

作っているうちに、広げ方をいくつか思いつきました。

- **お題を当てる、人狼みたいな遊びにする**。「AIアートインポスター」のように、何人かが同じお題でメイクをして、ひとりだけ違うお題。どれが紛れ込んでいるかを当てる、という形にしても盛り上がりそうです。
- **もっと"伝わる"ようにする**。いまは差が繊細すぎるので、濃さまで指定できるAPIを使えば、狙った気分がもう少しはっきり顔に出るはずです。難易度を選べるようにするのも良さそう。
- **自分の顔で参加する**。自分の写真に「今日の気分メイク」を乗せて投稿し、みんなで当て合う。あるあるは人それぞれなので、お題はいくらでも増えていきます。

メイクって、こんなふうにデータとして遊べるんだ、というのが個人的な発見でした。
