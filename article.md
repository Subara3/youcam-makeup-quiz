## 「地味ハロウィン」を、メイクでやりたかった

毎年たのしみにしている地味ハロウィン。「特に何でもない人」を全力で再現するあのノリが好きで、ずっと思っていました。**メイクでも、できないか?** と。

たとえば、会議中のこの人。

> 「微熱があるので場合によっては帰りたい」

血色をそっと一段抜いて、チークは置かない。リップはマットでくすませる。でも眉とアイラインは一応ある——だって、**まだ帰るとは言ってない**から。

この“さじ加減”、たぶん他人には**まったく伝わりません**。でも本人の中では精密に再現されている。その滑稽さをクイズにしました。

## キモは「同じシチュ × メイク差分」

地味ハロウィンの肝は **“同一人物・同一シーン”** であること。だから、用意したシチュ写真1枚に、感情ごとのメイクを乗せ分けて差分化しました。

![同じ会議中に4通りのメイク差分](screenshots/05_diff.png)

同じ会議室、同じ人。なのに今日の気分だけが違う——というのを、リップの色温度・チークの有無・眉の強さで表現しています。感情→メイクのマッピングはこんな方針（詳細は `PARAMS.md`）。

| 気分 | メイク差分の方針 |
| --- | --- |
| はしゃぎ | ピンクのグロス＋血色＋ラメ＋ハイライト強め |
| いらだち | ディープレッドのマット＋眉とアイラインを締める |
| しょんぼり | くすみリップ薄め＋チークを置かない＋ツヤ控えめ |
| ごきげん | コーラルのシアー＋ナチュラルな血色＋軽いツヤ |

## 量産は Playwright で自動化

YouCam のバーチャルメイクは、リップ／チーク／アイシャドウ／アイライナー／アイブロウ／ハイライトを、**色・濃さ・質感（マット/サテン/シアー/グロス/シマー…）** まで個別に指定できます。これが「細かすぎて伝わらない」を作るのにぴったりでした。

ただ手作業で 14シーン × 4気分 = 56枚は厳しいので、YouCam オンラインエディターを **Playwright で自動操作**。ポイントは2つ。

**① 目標の色に“いちばん近いスウォッチ”を選ぶ**

エディタのカラーパレットは、各色玉が `background-color` を持っています。これを読んで、狙った hex との距離が最小の玉をクリックすれば、`PARAMS.md` の設計に追従できます。

```js
const balls = [...document.querySelectorAll('[class*="color-ball_singleColor"]')];
let best, bd = 1e9;
for (const e of balls) {
  const [r, g, b] = getComputedStyle(e).backgroundColor.match(/\d+/g).map(Number);
  const d = (r - tr) ** 2 + (g - tg) ** 2 + (b - tb) ** 2;   // 目標色との距離
  if (d < bd) { bd = d; best = e; }
}
best.click();
```

**② DLは有料ゲートなので、結果プレビューを直接キャプチャ**

バーチャルメイクは透かしなしDLが有料。今回は無料枠＋透かしのまま使いたかったので、編集結果のプレビュー要素を `clip` 撮影しました。透かしは YouCam 製の証なので、記事用途ではむしろ好都合です。

さらに **ブラウザは1セッション開きっぱなし**にして、写真だけエディタ内で差し替え（毎回の再起動なし）。途中でタブが落ちても作り直して継続、既存ファイルはスキップ——で 56枚を一気に生成しました。

> 同じ `look` を makeup-vto **API**（`/s2s/v2.0/task/makeup-vto`、Bearer 認証）の effects 配列へ変換するスクリプトも入れてあります。色は `#RRGGBB`、濃さ 0–100、質感はテクスチャ名でそのまま渡せます（※APIユニットは有料）。
